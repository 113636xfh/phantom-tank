# phantom-tank
幻影图，双重图自动生成
使用方法：
1.将py文件下载到本地
2.创建new，in，out三个文件夹
3.把要当做表层的图片按顺序放入out，里层放入in
4.运行
5.程序会根据图片最后访问时间排序，把两个文件夹里排名相同的合成一张幻影图

plus：程序会自动调整输入图片大小，格式，最后输出png格式

我是在ubuntu调试的，windows可能需要一些调整
如果不能正常运行，可先将代码做如下调整，debug后改回来
